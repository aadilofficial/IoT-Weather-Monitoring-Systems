# IoT Weather Monitoring System

## Overview
The IoT Weather Monitoring System monitors environmental parameters such as temperature and humidity using sensors connected to an ESP8266/ESP32. The data is uploaded to the cloud for real-time monitoring.

## Features
- Real-time temperature monitoring
- Humidity monitoring
- Cloud data logging
- Wi-Fi connectivity
- Live dashboard support

## Hardware Required
- ESP8266(NodeMCU)
- DHT11 Sensor
- Breadboard
- Jumper Wires
- USB Cable
- Power Supply

## Software Required
- Arduino IDE
- Firebase / Blynk (if used)
- ESP8266 Board Package

## Folder Contents
- `Weather_Monitoring_Systems.ino` – Arduino source code
- `Weather_Monitoring_System_Schematic.png` – Circuit diagram
- `weather_monitoring_system.pdf` – Project documentation

## How to Run
1. Install Arduino IDE.
2. Install ESP8266 board package.
3. Install required libraries.
4. Update Wi-Fi credentials.
5. Upload the code to the ESP board.
6. Monitor the sensor data.

## Output
The system displays real-time temperature and humidity values and uploads them to the cloud.
